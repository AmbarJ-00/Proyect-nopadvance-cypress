# Informacion
Este proyecto tiene como objetivo hacer un recorrido sobre la aplicacion nodePad para verificar su funcionamiento, hacer un analisis de su estructura y detectar oportunidades de mejora, asi emmpleando las habilidades y conocimientos sobre la automatizacion de pruebas disponibles en mi persona.

## Integraciones
* Cypress.
* Grafana.
* Jenkins.
* Gherkin.
* Jmeter.
* Owasp Zap.

## Patrones de diseño
- Page object model(POM): es un patrón de diseño que crea un repositorio/carpeta de objetos para los elementos de la interfaz de usuario de una aplicación web.
Idea central: Cada página web de la aplicación se modela como una clase Java (u otro lenguaje).
